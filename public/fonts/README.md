# Local fonts

Put your **Kallsito** font file here so the job template header can use it.

**Supported filenames (use one):**

- `Kallsito.woff2` (preferred)
- `Kallsito.woff`
- `Kallsito.ttf`
- `Kallsito.otf`

**Steps:**

1. Copy the Kallsito font file from your laptop into this folder: `public/fonts/`
2. Rename it to `Kallsito.woff2`, `Kallsito.ttf`, or `Kallsito.otf` to match your file type
3. Restart the dev server if it’s running
